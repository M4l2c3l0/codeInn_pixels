Attributes

We now know that an HTML element is defined by generally (we will go through the exceptions on the following section) a couple of HTML tags along with the content in-between them, but as we are now about to see, some significant and rather useful alterations can be applied to it by the use of attributes.
Although not always required, attributes can add very useful and specific information to elements, so much so that multiple ones are often used within a single element.

Syntax

Attributes must always be placed inside the opening tag, separated by at least one whitespace from the tag name, and before the closing angle-bracket >.
Its syntax consists of its name typed in lower-case, directly followed by the = sign (no whitespace required), and completed by the corresponding value in-between two quotation marks "" or '' (whitespace also not required).

In order to place additional attributes within the same tag, simply separated each one by at least one whitespace:
<tag attr="value" attr="value">

Useful Attributes

lang although not used within our html code sample, the lang attribute defines the spoken language of your document, and is placed within the opening html tag:
<html lang="en">
id provides a unique id exclusively referring to that element, meaning that when using multiple ids within the same HTML document, all must be unique and cannot be given to multiple elements:
<div id="unique-id">
class unlike an id, a class attribute provides an identifier that may be given to multiple elements, allowing a group of different elements sharing the same class value to be all targeted at once:
<h1 class="article1">
<p class="article1">
href using the example of the hyperlink tag (<a>), the href attribute is what provides the url adress where the user is to be redirected upon clicking on it:
<a href="url">
src the src attribute provides the directory to a document, which is to be uploaded and displayed along with the webpage:
<img src="directory">
title gives a title to an element, which is displayed several seconds after placeing the mouse over it:
<img title="description">
alt defines a description to an image which is displayed in case the image fails to load, and is good practise to use as it assists people with visual-impairment by spelling it:
<img alt="this is an image">
