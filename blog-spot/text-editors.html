Text Editors

What text editor should I chose?
Which is the best one?
Well, as I understand it, each was designed to fullfil specific tasks, therefore do not bother searching for the best one, but rather the most suitable for your needs, which in our case is web design. So to give you some food for thought, I will start by going through some of the free options I have personally tried, and finish by a short list of a few excellent other choices that I am yet to try.

TextWrangler

Price: Free
Platform(s): Mac OS X
Developer: Bare Bones Software
Website: barebones.com/products/textwrangler/

TextWrangler is an all-purpose text and code editor for Mac OS X, based on the same award-winning technology as BBEdit, our leading professional HTML and text editor.
(source: barebones.com)

When starting to learn code, the temptation of picking an editor that does a lot of work for you is high, and after all why wouldn't you?
One of the best tips I was given and could share with you, is to be alright at walking before trying to run. If like me you would like to make sure you are able to code all on your own before trying out another editor that will reduce and chew a lot of your work for you, then all you need for now is a simple and clear editor such as TextWrangler with just the basic features that really matter.

Brackets

Price: Free
Platform(s): Mac OS X/ Linux / Windows
Developer: Adobe Systems
Website: brackets.io

With focused visual tools and preprocessor support, Brackets is a modern text editor that makes it easy to design in the browser. It's crafted from the ground up for web designers and front-end developers.
(source: brackets.io)

Seriously pumped-up with great features such as live preview, which displays in real-time any HTML or CSS changes (requires Google Chrome), or inline editors allowing you to style elements from the HTML document without switching files, it is to my knowledge one of the most web design oriented text editors.
So although I did not rant nor rate any of the text editors in this article, Brackets has to be one of my highest recommendations for front-end developers, and one I have been using ever since discovering it.

Atom

Price: Free
Platform(s): Mac OS X / Linux / Windows
Developer: GitHub Inc.
Website: atom.io

Atom is a text editor that's modern, approachable, yet hackable to the core â€” a tool you can customise to do anything but also use productively without ever touching a config file.
(source: atom.io)

Developed by GitHub, which for those of you (like me) who are not too familiar with, seems to be home to a HUGE open-source community, Atom consequently has an astonishing amount of themes and extension packages allowing you to personalise the editor to your every need. What's more, if you are still unable to find an extension or theme you'd like, Atom basically allows you to code it yourself!
With such potential and flexibility, and although it is not necessarely a web-development oriented editor to start with, Atom really seems to appeal to all kind of coders, and since briefly trying it out, one I look forward to get more familiar with.

Other Great Editors

  • Coda

    Another amazing web design oriented editor (Mac OS X only). Priced at $99 USD, it offers great multiple websites control, has a built-in preview, and a whole lot more amazing features, making it seriously worth considering.
    website: panic.com/coda/

  • NotePadd++

    Free editor (Windows only), it has been around since 2003 and remains widely used amongst coders, confortable with its simplistic yet efficient numerous features.
    website: notepad-plus-plus.org

  • Sublime

    Its price tag of $70 USD (free trial available) clearly seems to be justified by its highly acclaimed performance, customizable interface, and incredible community support. Hands down one of the most popular worldwide.
    website: sublimetext.com
