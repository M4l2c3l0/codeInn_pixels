Display Value

Although our view on HTML elements has mainly been focus around the code so far, let's wrap-up the HTML chapter with a different but just as important perspective on them: the layout impact.
All elements have a display value which in turn has two values: block or inline, and is defined by their type (heading, paragraph, image...).

So let's dive right into our last HTML section in order to understand a very important point with regards to our HTML elements' layout, and try it out on some of the elements we are now familiar with.

Inline vs. Block

Block elements always start on a new line and unless instructed differently, will take all the available width from left to right of their parent element.
They directly follow each other on the same vertical order than the one used on the HTML document, and shift to the left side by default (this is only visible if they are defined with a specific width and not centered as we will later see in the CSS chapter).
A block element is also able to contain an inline element within its content, which does not work the other way around.

Inline elements on the other hand do not require to start on a new line, simply following others from left to right, and can be nested inside block elements.
Rather than using all available width, theirs is just long enough to fit their content's length, and break down to the following line only if not enough space is available on the current one.
Above for example, notice that the second element (first line) and the fifth (third line) carry over to the following line, for not enough space was available on their respective ones.

Block or Inline?

<h1> - <h6> any heading has a block value.
<p> a paragraph has a block value.
<img> an image has a block value.
<a> an anchor has an inline value, often included within more text.
<input> the input has a block value.
<div> a div has a block value, generally used to next others within its content.
<br> although breaking onto a new line, the line-break has an inline value as it is intended to be placed within some text.
