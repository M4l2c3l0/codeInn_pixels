A Bit Of History

Imagined by the mind of physicist Tim Berners-Lee in his ENQUIRE communication system (described as the predecessor to the World Wide Web) for CERN researchers where he worked as a contractor in the 80s, HTML soon became a standard for webpage creation since its release in the early 90s.
In a nutshell, HTML is used to create a webpage's structure by the use of markup tags (or HTML tags here), which are then read, interpreted, and displayed by the web browser. Consequently, HTML is regarded as a markup language rather than a programming one.
Following several new versions throughout the years, the HTML5 release was official made in 2014, and is to date the most recent update.

First Impression

<!doctype html>
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    <h1>Big Heading</h1>
    <p>This is a paragraph.</p>
  </body>
</html>

Simply put, HTML files are text files completed by the .html extension, consisting of a list of HTML markup tags.
As shown above, an HTML tag is made of a specific keyword or letter defining its type (title, heading, paragraph, ...), wrapped by two angle brackets <>, and in most cases followed by its exact copy with an extra / before its name.
As we are about to find out in more detail, the combination of both tags forms an HTML element.
